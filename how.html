How I Structured My Webpage Using HTML Elements

• Document Type Declaration for HTML5.
• HTML Root Element with specified language attribute.
• Head Section containing metadata and external resource links.
• Character Encoding specification.
• Viewport for Responsive Design definition.
• Document Title setting.
• Linking to Bootstrap CSS.
• Preconnection to Google Fonts and linking to the Nunito font.
• Embedded CSS Styles.
• CSS Styles for:
    • Body, including font-family, opacity, and animation.
    • Navbar, with custom styles for background, border-radius, padding, and box-shadow.
    • Navbar brand/logo.
    • Navigation items and links, including hover effects.
    • History Section, with styling for a top margin.
    • Card element, covering appearance, transition effects, and layout styles.
    • Card hover effects, including a transform and background color change.
    • Top image in the card.
    • Different elements within the card, like card-body, card-title, card-text, and read-more.
